Projects

cChannels - HackMoney Compound Prize Winner

Interest earning payment channels
Description: cChannels are a series of payment channel contracts that support both Ether and ERC20 payments. When the funds are deposited into a cChannel, they are then locked into Compound and are constantly earning interest for the depositor, while being spent off chain. cChannels won the Compound bounty during the HackMoney hackathon.

Tools: React, Web3, Solidity, EIP712, EIP1167, eth_signTypedData_v3

Peer Lending - EthDenver 2020

A get a loan now, not later.
Description: Worked as a team to build a peer-to-peer lending tool that allows people to create micro loan fundraisers at Eth Denver Hackathon. I specifically worked on the web3 code that helped pipe data between the frontend and smart contract.

Tools: React, Web3.js, Truffle, Solidity, OpenZeppelin Contracts, Rebass, BN.js

CryptoKitty Playpen

A place to play with your cryptokittes.
Description: The CryptoKitty Playpen allows users to view and interact with their cryptokitties. Feeding the kitties a fish will make them jump for joy. Users can also transfer, wrap, or unwrap their cryptokitties.

Tools: React, Web3.js, OpenSea API, React Spring, ERC721 Standard, Wrapped CryptoKitties

Set Factory

Creating a token set has never been easier.
Description: A user interface that allows user to create a Stable Token Set using the Set Protocol.

Tools: React, Redux, web3.js, D3, Styled Components

CryptoClicker

The cryptoeconomy is just a click away.
Description: Similar to Cookie Clicker, CryptoClicker is an incremental game where users can score points by clicking. However, CryptoClicker allows users to turn their in-game currency into Ethereum ERC20 tokens. As a result, cryptoclicker is a proof of concept of how a game could reduce friction for players trying to sell their in-game currency on secondary markets e.g. selling Runescape or WoW gold.

Tools: React, Redux, Node.js, Solidity, Web3.js, Truffle, Remix, ERC20 Token Standard, OpenZeppelin Contracts, SQL, CSS3

King of the Dapp

Lock up Ether to become the new King.
Description: A simple overview on how to integrate a React interface with an Ethereum smart contract posted on Medium. King of the Dapp is a status game where users lock up Ether in a contract in order to be crowned King.

Tools: React, Solidity, Web3.js, Truffle, Mocha, OpenZeppelin Contracts, CSS3
